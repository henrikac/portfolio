Portfolio

As mentioned before, I really love to make web scrapers and task automation.
One of the more interesting projects that I have done is my Instagram bot (SocialBots). I made this project as an experiment to see if I would get more followers on Instagram if I just liked a bunch of random photos. I was not interested in finding and liking all the photos myself so I decided to automate this task using Selenium. The project turned out to be a success because the more photos my bot liked the more followers I got.

I created a few JavaScript projects while following the curriculum on FreeCodeCamp. Those projects of course varied in difficulty, from simple animations to fetching data from an API. The most difficult project, in my opinion, was the Simon Game - not because of the JavaScript but because of the styling part.
